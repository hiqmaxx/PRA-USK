APLIKASI INVENTARIS BARANG KANTOR

Deskripsi:
Aplikasi ini merupakan aplikasi web sederhana berbasis Single Page Application (SPA) yang digunakan untuk mencatat data inventaris barang kantor. Aplikasi ini dibuat menggunakan HTML, CSS (Bootstrap), dan JavaScript.

Fitur:
1. Menambahkan data barang (nama barang, jumlah, kategori)
2. Menampilkan daftar barang dalam bentuk tabel
3. Menghapus data barang
4. Validasi input (tidak boleh kosong)

Struktur Folder:
- index.html  : Tampilan utama aplikasi
- style.css   : Styling tambahan
- script.js   : Logika aplikasi
- README.txt  : Dokumentasi

Cara Menjalankan Aplikasi:
1. Buka folder project
2. Jalankan file index.html menggunakan browser
   atau menggunakan Live Server di Visual Studio Code
3. Aplikasi siap digunakan

Cara Penggunaan:
1. Isi form tambah barang
2. Klik tombol "Simpan Barang"
3. Data akan muncul di tabel
4. Klik tombol "Hapus" untuk menghapus data

Teknologi yang Digunakan:
- HTML
- CSS (Bootstrap 5)
- JavaScript

Catatan:
Data yang dimasukkan hanya disimpan sementara (tidak menggunakan database). Jika halaman di-refresh, data akan hilang.
